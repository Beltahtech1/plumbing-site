# Beltah Plumbing AI Website

![Beltah Plumbing Logo](https://files.catbox.moe/jt2av2.jpg)

A premium, modern React + Tailwind website for **Beltah Plumbing** in Nairobi.  
This project integrates a **Gemini AI Diagnostic Tool** to provide instant plumbing advice, alongside a sleek responsive design and 24/7 emergency contact features.

---

## 🚀 Features
- **Modern UI** built with React + Vite + TailwindCSS
- **Custom Logo** (see above)
- **High-contrast blue & orange palette**
- **24/7 Emergency Section** with quick contact info
- **Service Cards** showcasing plumbing offerings
- **AI Diagnostic Tool** powered by Gemini API
- **Lucide-react Icons** for clean visuals
- **WhatsApp/Call Dropdown Menu** for instant communication
- **Fully Responsive** with smooth scroll animations
